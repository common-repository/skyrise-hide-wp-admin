== Description ==

Skyrise Hide WP Admin is a plugin for WordPress, which hides the admin panel. It allows you to save a website from a planned hacker attack. It hides and protects the administrator panel, while the classic wp-admin address redirects to 404, everyone who is not logged in.

There is always the danger that you may experience some incident involving attempts to break into your CMS, which means you may lose your data and website. Skyrise Hide WP Admin provides your website basic, required security.


== Frequently Asked Questions ==

= How to use =

For proper operation of this plugin you should install this, go to = `Settings` -> `General` = and set the redirect url (by default `404`)
